# Personal Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. It features a bold hero section, project showcase, skills highlights, a contact form, and subtle scroll-reveal animations.

## Features
- Responsive layout with mobile navigation
- Hero + About + Projects + Skills + Contact sections
- Scroll reveal animations for sections and cards
- Lightweight and fast (no frameworks)
- Contact form with a simple client-side message

## Built With
- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started
1. Clone or download the repo.
2. Open `index.html` in your browser.

## Customization
- Update text content in `index.html`
- Adjust styles in `style.css`
- Edit animations and interactions in `script.js`

## Project Structure
- `index.html` – page structure and content
- `style.css` – styling and layout
- `script.js` – navigation toggle, reveal animations, and form behavior
